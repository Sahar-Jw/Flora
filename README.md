# Flora - Real Estate Landing Page

## Overview
Flora is a modern, responsive landing page for a real estate platform designed to help users discover and book their dream homes. It features a clean, visually appealing interface focused on showcasing apartments and properties, primarily targeting locations like Ahmedabad, India.

## Purpose
The website serves as a promotional front-end for real estate services, allowing users to:
- Explore trending properties and luxury apartments (e.g., priced $1,000–$10,000).
- Browse featured real estate deals across categories like Residential, Commercial, Agriculture, and Industrial properties.
- Follow a simple 3-step process: **Search Apartment** → **Select Apartment** → **Confirm Apartment**.

## Key Features
- **Hero Section**: Eye-catching banner with property details (location: Ahmedabad, India; type: Apartment; price range).
- **How It Works**: Step-by-step guide with icons for property search, selection, and confirmation.
- **Find Dream Home**: Full-width promotional section with overlay text.
- **Most Trending**: Grid of 6 property cards with prices, titles (e.g., \"Luxury Apartment in California\"), and locations.
- **Best Real Estate Deals**: Filterable buttons for property types and featured cards with \"Featured\" and \"3D\" badges.
- **Navigation**: Fixed header with links (Home, About, Service, New Property, Contact) and Login button.
- **Footer**: Company info, services, about links, location, social media (Facebook, Twitter, LinkedIn), and legal links.
- **Responsive Design**: Fully mobile-friendly with hamburger menu on smaller screens.

## Technologies Used
- **HTML5**: Semantic structure for header, main sections, nav, footer, and interactive elements (buttons, links).
- **CSS3**: Custom styling with:
  - Flexbox and CSS Grid for layouts.
  - CSS Variables (e.g., `--primary-color: #025595`).
  - Backdrop filters, box-shadows, hover animations, and transitions.
  - Media queries for responsive breakpoints (e.g., max-width: 992px for mobile).
  - Global reset (`* { margin: 0; padding: 0; box-sizing: border-box; }`).
- **Google Fonts**: Inter (primary), Karla, Nunito for typography.
- **SVG Icons**: Custom SVGs for search, select, confirm steps, location, house, dollar icons.
- **Images**: High-quality JPGs/PNGs/SVGs for hero background, property thumbnails, logos, social icons.

This static single-page site is optimized for quick loading and can be easily hosted on GitHub Pages or any static host. Open `index.html` in a browser to preview!

To view the site: `start index.html` (opens in default browser on Windows).
